# :shield: Project Monroe :candle:

Project monroe is a game made by the team All Nerds Unite Secretly, a group of 4 motivated students in their first year of preparatory class at Epita.

The game features a rogue-like gameplay with procedurally generated dungeon, 
that one must explore in order to retrive the losts artefacts and hope to break the curse. 
You may gather items that serve as abilities, allowing you to customize your moveset
